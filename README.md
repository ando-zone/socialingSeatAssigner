# 🎯 모임 자리 배치 프로그램 (Socialing Seat Assigner)

모임, 워크샵, 네트워킹 이벤트에서 참가자들을 최적으로 그룹 배치하는 스마트한 웹 애플리케이션입니다.

## ✨ 주요 기능

### 🧠 스마트 그룹 배치 알고리즘
- **이전 만남 회피**: 이전 라운드에서 만난 사람들을 자동으로 회피
- **성별 균형**: 각 그룹의 성별 비율을 최적화
- **MBTI 균형**: 외향형과 내향형의 균형있는 배치
- **그룹 히스토리 관리**: 연속된 라운드에서 같은 그룹 번호 회피

### 👥 참가자 관리
- **개별 추가**: 이름, 성별, MBTI 정보로 참가자 등록
- **벌크 추가**: 여러 명을 한번에 추가 (CSV, 공백 구분 등 다양한 형식 지원)
- **참가자 삭제**: 중도 이탈자 관리 (이탈자 정보는 통계에 반영)

### 🎮 유연한 그룹 설정
- **자동 모드**: 동일한 크기의 그룹으로 자동 배치
- **수동 모드**: 각 그룹별로 개별 인원 수 설정 가능

### 📊 상세한 분석 및 통계
- **실시간 통계**: 성별/MBTI 균형 점수, 새로운 만남 수
- **히스토그램**: 전체 만남 수, 이성 만남 수 분포
- **개별 추적**: 각 참가자의 만남 히스토리 상세 조회

### 🔄 실시간 조정
- **드래그 앤 드롭**: 참가자를 끌어다 놓아 위치 변경
- **즉시 업데이트**: 변경 시 모든 통계와 만남 기록 자동 재계산
- **신규 참가자 추가**: 각 그룹에 현장에서 참가자 추가 가능

### 💾 데이터 백업 및 복원
- **자동 스냅샷**: 주요 작업 시 자동으로 상태 저장
- **JSON 내보내기/가져오기**: 데이터 백업 및 복원
- **시점 복원**: 특정 시점으로 되돌리기 가능

## 🚀 설치 및 실행

### 필수 요구사항
- Node.js 18.17 이상
- npm 또는 yarn

### 설치
```bash
# 저장소 클론
git clone https://github.com/dohyeonan/socialingSeatAssigner.git
cd socialingSeatAssigner

# 의존성 설치
npm install
```

### 개발 서버 실행
```bash
npm run dev
```
http://localhost:3000에서 애플리케이션에 접근할 수 있습니다.

### 프로덕션 빌드
```bash
npm run build
npm start
```

### GitHub Pages 배포
```bash
npm run deploy
```
정적 사이트로 빌드되어 `out/` 폴더에 생성됩니다.

## 📖 사용법

### 1. 참가자 등록
- **개별 등록**: 이름, 성별, MBTI를 입력하여 한 명씩 추가
- **벌크 등록**: "여러 명 한번에 추가" 버튼으로 대량 등록
  ```
  지원 형식:
  김철수
  이영희,여,내향
  박민수 남 외향
  최지은 여 내향
  ```

### 2. 그룹 설정
- **자동 모드**: 그룹 크기(3-6명) 선택 → 자동으로 그룹 수 계산
- **수동 모드**: 그룹 수와 각 그룹별 인원 수 개별 설정

### 3. 그룹 배치 실행
- "그룹 배치하기" 버튼 클릭
- 알고리즘이 최적 배치 계산 후 결과 페이지로 이동

### 4. 결과 확인 및 조정
- **그룹 결과 탭**: 각 그룹의 구성원과 균형 지표 확인
- **참가자 통계 탭**: 만남 분포 히스토그램과 개별 기록 조회
- **실시간 조정**: 드래그 앤 드롭으로 참가자 위치 변경
- **추가 등록**: 각 그룹에 현장 참가자 추가

### 5. 다음 라운드 준비
- "다음 라운드 준비" 버튼으로 새로운 라운드 시작
- 이전 라운드 만남 기록이 자동으로 보존됨

## 📊 알고리즘 설명

### 배치 최적화 기준
1. **만남 다양성**: 이전에 만나지 않은 사람들과 우선 배치
2. **성별 균형**: 각 그룹 내 성별 비율 균형
3. **MBTI 균형**: 외향형과 내향형의 적절한 분배
4. **그룹 번호 회피**: 연속 라운드에서 동일 그룹 번호 회피

### 점수 계산
- **성별 균형 점수**: `1 - |남성수 - 여성수| / 총원` × 100
- **MBTI 균형 점수**: `1 - |외향형수 - 내향형수| / 총원` × 100
- **80점 이상**: 우수, **60-79점**: 보통, **60점 미만**: 개선필요

## 🔧 고급 기능

### 백업 및 복원
- **자동 스냅샷**: 참가자 추가/제거, 그룹 배치, 위치 변경 시 자동 저장
- **수동 백업**: JSON 파일로 모든 데이터 내보내기
- **데이터 가져오기**: 이전에 내보낸 JSON 파일 불러오기
- **시점 복원**: 특정 스냅샷으로 되돌리기

### 실시간 조정
- 드래그 앤 드롭으로 참가자 위치 변경
- 변경 시 모든 통계와 만남 기록 자동 재계산
- 같은 그룹 내 위치 변경 방지

### 통계 분석
- 개별 참가자 만남 기록 상세 조회
- 전체/이성 만남 수 히스토그램
- 라운드별 진행 상황 추적

## 🎯 활용 사례

- **네트워킹 이벤트**: 참가자들이 다양한 사람들과 만날 수 있도록
- **워크샵/세미나**: 토론 그룹 구성
- **팀 빌딩**: 부서간 교류 활동
- **컨퍼런스**: 점심 식사 테이블 배치
- **스터디 그룹**: 다양한 조합의 학습 팀 구성
