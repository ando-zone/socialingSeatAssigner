# 🎪 소셜링 자리 배치 프로그램 (Socialing Seat Assigner)

모임에서 참가자들이 매 라운드마다 새로운 사람들과 만날 수 있도록 최적화된 자리 배치를 제공하는 프로그램입니다.

## 🎯 프로젝트 목표

- **새로운 만남 극대화**: 매 라운드마다 이전에 만나지 않았던 사람들끼리 그룹을 구성
- **성별 균형 고려**: 각 그룹의 성별 비율을 균형있게 배치
- **최적화 알고리즘**: 유전 알고리즘을 사용하여 최적의 그룹 조합 찾기
- **실시간 분석**: 이론적 한계와 실제 진행 상황을 실시간으로 분석

## ⚙️ 주요 고려사항

### 1. 유연한 그룹 크기 설정
- **그룹마다 최대 인원이 다를 수 있음**: 현재는 균일한 그룹 크기로 구현되어 있지만, 향후 각 그룹별로 다른 최대 인원을 설정할 수 있도록 확장 예정
- 예시: 라운드 테이블 크기가 다르거나, 특정 활동에 따라 그룹 크기가 제한되는 경우

### 2. 이전 만남 회피 최적화
- **가능한 이전에 속한 그룹에 속하지 않는 것을 시도**: 알고리즘이 우선적으로 새로운 만남을 극대화하도록 설계
- 기존 만남 기록을 바탕으로 중복 배치를 최소화
- 완전히 새로운 만남이 불가능한 경우에만 이전에 만난 사람들과 재배치

### 3. 성별 균형 고려
- 각 그룹 내 성별 비율을 가능한 균등하게 유지
- 전체 참가자 성비에 따라 자동으로 이상적인 비율 계산

## 🎲 알고리즘 동작 원리

### 점수 계산 시스템
1. **기존 만남 페널티**: 이전에 만난 적 있는 쌍마다 +1점
2. **성별 불균형 페널티**: 이상적인 성비에서 벗어날 때마다 +2점
3. **목표**: 총 점수를 최소화 (0점이 완벽한 해답)

### 유전 알고리즘
1. 참가자들을 랜덤하게 섞어서 그룹 생성
2. 각 조합의 점수 계산
3. 최대 1,000번 반복하여 최적해 탐색
4. 완벽한 해답(점수 0)을 찾으면 조기 종료

## 🔢 수학적 근거

N명을 n명씩 그룹으로 나눌 때:
- **전체 가능한 만남**: C(N, 2) = N×(N-1)÷2
- **라운드당 소모되는 만남**: (N÷n) × C(n, 2)
- **이론적 최대 라운드**: (N-1) ÷ (n-1) (근사치)

예: 72명을 12명씩 → 약 71÷11 = 6.45라운드 → 실제로는 6라운드 가능
